Clase 10: Ejercicios de Introducción a los Atributos Globales

Ejercicios Prácticos

Aplica tus conocimientos sobre los atributos globales en HTML.

Ejercicio 1: Usando id y class

Crea un <div> con el id="contenedor-principal". Dentro de este <div>, crea tres párrafos. El primero debe tener la class="parrafo-rojo". El segundo, la class="parrafo-azul". El tercero debe tener ambas clases: "parrafo-rojo" y "parrafo-azul".

<div id="contenedor-principal">
    <p class="parrafo-rojo">Este párrafo es rojo.</p>
    <p class="parrafo-azul">Este párrafo es azul.</p>
    <p class="parrafo-rojo parrafo-azul">Este párrafo es rojo y azul.</p>
</div>

<!-- Estilos CSS de ejemplo (podrían ir en common_styles.css o en un <style> en el head) -->
<style>
    .parrafo-rojo {
        color: red;
    }
    .parrafo-azul {
        color: blue;
    }
</style>

Ejercicio 2: Atributos style y title

Crea un encabezado <h2> con el texto "Mi Título Especial". Aplícale un estilo en línea para que el texto sea de color morado y tenga un tamaño de fuente de 2em. Añade un title que diga "Este es un título muy importante".

<h2 style="color: purple; font-size: 2em;" title="Este es un título muy importante">Mi Título Especial</h2>

Ejercicio 3: Atributos lang y dir

Configura la etiqueta <html> de la página para que el idioma principal sea español. Luego, dentro del <body>, crea un párrafo con contenido en inglés ("This is a paragraph in English.") y otro párrafo con contenido en árabe ("هذا نص باللغة العربية.") asegurándote de usar los atributos lang y dir correctos para cada uno.

<!DOCTYPE html>
<html lang="es"> <!-- Idioma principal de la página: español -->
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ejercicio de Atributos Globales</title>
</head>
<body>
    <p>Este es un párrafo en español.</p>
    <p lang="en">This is a paragraph in English.</p>
    <p lang="ar" dir="rtl">هذا نص باللغة العربية.</p>
</body>
</html>
