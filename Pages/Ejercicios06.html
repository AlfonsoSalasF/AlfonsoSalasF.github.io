Clase 6: Ejercicios de Enlaces (Hyperlinks)

Ejercicios Prácticos de la Clase 6

Para consolidar tus conocimientos sobre enlaces, te propongo los siguientes ejercicios. Haz clic en "Resolver" para ver la solución de cada uno.

Ejercicio 1: Navegación entre páginas

Crea una página de "Inicio" (index.html) y una de "Servicios" (servicios.html).

  • En index.html, crea un enlace que lleve a servicios.html.
  • En servicios.html, crea un enlace que lleve de vuelta a index.html. Utiliza rutas relativas.

Solución Ejercicio 1:

Contenido de index.html:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Inicio</title>
</head>
<body>
    <h1>Página de Inicio</h1>
    <p>Ir a <a href="servicios.html">Nuestros Servicios</a>.</p>
</body>
</html>

Contenido de servicios.html:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Servicios</title>
</head>
<body>
    <h1>Nuestros Servicios</h1>
    <p>Volver a la <a href="index.html">Página de Inicio</a>.</p>
</body>
</html>

Ejercicio 2: Enlace externo con nueva pestaña y seguridad

Añade un enlace externo en tu página de "Inicio".

  • Enlaza a un sitio web de noticias o una enciclopedia (ej. Wikipedia) que se abra en una nueva pestaña y asegúrate de incluir las medidas de seguridad recomendadas (rel="noopener noreferrer").

Solución Ejercicio 2 (ejemplo en index.html):

<p>Visita la <a href="https://es.wikipedia.org/wiki/HTML" target="_blank" rel="noopener noreferrer">página de HTML en Wikipedia</a> (se abrirá en una nueva pestaña de forma segura).</p>

Ejercicio 3: Implementar enlaces de contacto

Implementa enlaces de contacto en tu página de "Servicios".

  • En tu página de "Servicios", añade un enlace de correo electrónico para "Preguntas" que pre-rellene el asunto como "Consulta sobre servicios".
  • Añade un enlace de teléfono (con un número ficticio) para dispositivos móviles, visiblemente formateado.

Solución Ejercicio 3 (ejemplo en servicios.html):

<p>¿Preguntas? <a href="mailto:info@tudominio.com?subject=Consulta%20sobre%20servicios">Envíanos un correo</a>.</p>
<p>Llámanos: <a href="tel:+34912345678">+34 912 34 56 78</a>.</p>

Ejercicio 4: Crear un menú de navegación básico

En tu index.html, crea una lista desordenada (<ul>) con elementos de lista (<li>). Dentro de cada <li>, coloca enlaces a:

  • "Inicio" (a index.html mismo)
  • "Servicios" (a servicios.html)
  • "Contacto" (un enlace mailto: o un enlace a una sección #contacto si la añades).

Solución Ejercicio 4 (ejemplo en index.html):

<nav>
    <ul>
        <li><a href="index.html">Inicio</a></li>
        <li><a href="servicios.html">Servicios</a></li>
        <li><a href="mailto:info@tudominio.com">Contacto</a></li>
    </ul>
</nav>

Ejercicio 5: Enlaces a secciones en la misma página (Desafío)

En tu index.html, crea al menos tres secciones (por ejemplo, "Acerca de", "Nuestros Valores", "Testimonios") y dales un id único a cada una. En la parte superior de index.html, crea un pequeño menú de navegación con enlaces que te permitan saltar a cada una de estas secciones dentro de la misma página.

Solución Ejercicio 5 (ejemplo en index.html):

<!-- Menú de navegación -->
<nav>
    <ul>
        <li><a href="#acerca-de">Acerca de</a></li>
        <li><a href="#valores">Nuestros Valores</a></li>
        <li><a href="#testimonios">Testimonios</a></li>
    </ul>
</nav>

<!-- Secciones con IDs -->
<section id="acerca-de">
    <h2>Acerca de Nosotros</h2>
    <p>Contenido sobre la empresa...</p>
</section>

<section id="valores">
    <h2>Nuestros Valores</h2>
    <p>Contenido sobre los valores...</p>
</section>

<section id="testimonios">
    <h2>Testimonios</h2>
    <p>Contenido de testimonios...</p>
</section>

Preguntas Tipo Test

Pon a prueba tus conocimientos sobre enlaces HTML con estas preguntas de opción múltiple.

1. ¿Cuál es el atributo principal de la etiqueta <a> para especificar el destino del enlace?

2. Para abrir un enlace en una nueva pestaña del navegador, ¿qué atributo y valor debes añadir?

3. ¿Qué se recomienda añadir siempre junto con target="_blank" por motivos de seguridad y rendimiento?

4. ¿Cómo se crea un enlace para enviar un correo electrónico?

5. ¿Cuál es el formato correcto para un enlace telefónico que un dispositivo móvil pueda reconocer?

6. Si quieres enlazar a un archivo llamado pagina2.html que está en el mismo directorio que tu archivo actual, ¿cuál es la ruta correcta?

7. Para enlazar a una sección específica dentro de la misma página, ¿qué se usa en el atributo href?

8. ¿Qué tipo de ruta es /carpeta/archivo.html?

9. ¿Cuál de las siguientes URL apunta a un enlace externo?

10. Si un enlace tiene href="documento.pdf", ¿qué tipo de recurso esperaría el navegador?

11. ¿Para qué sirve el atributo rel="noreferrer"?

12. ¿Qué caracteres se utilizan para separar parámetros en una URL mailto: (después del primero)?

13. Si estás en blog/articulo.html y quieres enlazar a index.html en la raíz del sitio, ¿cuál es la ruta relativa correcta?

14. ¿Qué es un "ancla" en el contexto de los enlaces HTML?

15. ¿Es necesario incluir http:// o https:// para un enlace interno si se usan rutas relativas?

16. ¿Qué atributo permite pre-rellenar el asunto de un correo electrónico en un enlace mailto:?

17. ¿Cuál es la ventaja principal de usar rutas relativas para enlaces internos?

18. ¿Qué significa el texto entre las etiquetas <a> y </a>?

19. Si quieres que un enlace mailto: incluya un destinatario en copia oculta (BCC), ¿qué parámetro usarías?

20. ¿Qué se utiliza para indicar "subir un nivel en la estructura de directorios" en una ruta relativa?
