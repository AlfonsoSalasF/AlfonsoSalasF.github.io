Ejercicios Prácticos

Aplica tus conocimientos sobre la estructura básica de HTML resolviendo los siguientes ejercicios.

Ejercicio 1: Tu Primera Página HTML

Crea un archivo HTML con la estructura básica aprendida en la Clase 2. El título de la página debe ser "Mi Primera Página Web" y en el cuerpo de la página debe aparecer el texto "¡Hola, mundo!".

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Primera Página Web</title>
</head>
<body>
    <p>¡Hola, mundo!</p>
</body>
</html>

Ejercicio 2: Configurando el Idioma y los Metadatos

Modifica la estructura básica de un documento HTML para que el idioma principal del documento sea el inglés (en) y añade una etiqueta <meta> con la descripción "Página de ejemplo para el curso de HTML" (name="description").

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Página de ejemplo para el curso de HTML">
    <title>English Example Page</title>
</head>
<body>
    <p>Hello, world! This page is in English.</p>
</body>
</html>
