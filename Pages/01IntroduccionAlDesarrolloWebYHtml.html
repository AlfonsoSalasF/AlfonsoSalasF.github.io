Clase 1: Introducción al Desarrollo Web y HTML.

¡Bienvenido a la primera clase de nuestro curso de iniciación a HTML! Si estás aquí, es porque tienes curiosidad por saber cómo funcionan las páginas web y cómo puedes empezar a crearlas tú mismo. 1.1. ¿Qué es el Desarrollo Web? Frontend vs. Backend.

El desarrollo web es el proceso de crear sitios web y aplicaciones web. Es un campo vasto que se puede dividir en dos áreas principales:

1.1.1. Frontend (El Lado del Cliente)

El frontend se refiere a todo lo que el usuario ve y con lo que interactúa directamente en una página web. Es la parte "visible" de un sitio web, la interfaz de usuario. Los desarrolladores frontend se centran en la experiencia del usuario (UX) y la interfaz de usuario (UI).

Las tecnologías principales del frontend son:

  • HTML (HyperText Markup Language): La estructura y el contenido de la página web. Es el esqueleto.
  • CSS (Cascading Style Sheets): El estilo y la presentación visual de la página (colores, fuentes, diseño, etc.). Es la piel y la vestimenta.
  • JavaScript: La interactividad y la lógica del lado del cliente (animaciones, validación de formularios, etc.). Es el sistema nervioso y el cerebro que permite el movimiento.

Cuando navegas por una página web, todo lo que ves y haces en tu navegador es resultado del trabajo del frontend.

1.1.2. Backend (El Lado del Servidor)

El backend, por otro lado, es la parte "detrás de escena" de un sitio web. Incluye la lógica del servidor, la base de datos y la gestión de la aplicación. Es lo que permite que el frontend funcione, procesando las peticiones de los usuarios, almacenando y recuperando datos, y asegurando la seguridad y el rendimiento del sitio.

Las tecnologías comunes del backend incluyen lenguajes de programación como Python, Node.js, PHP, Ruby, Java, así como bases de datos (MySQL, PostgreSQL, MongoDB) y servidores web (Apache, Nginx).

Un desarrollador backend se asegura de que los datos se almacenen de forma segura, que las operaciones se realicen de manera eficiente y que la aplicación funcione correctamente en segundo plano.

En resumen: El frontend es lo que ves y con lo que interactúas en tu navegador; el backend es lo que hace posible que esa interacción funcione, manejando la lógica y los datos en el servidor.

1.2. El Papel de HTML en la Creación de Páginas Web.

HTML, o **Lenguaje de Marcado de Hipertexto**, es la base fundamental de cualquier página web. No es un lenguaje de programación, sino un lenguaje de marcado. Esto significa que utiliza "etiquetas" (tags) para estructurar el contenido de una página, definiendo qué es cada parte de texto (un encabezado, un párrafo, una lista, un enlace, una imagen, etc.).

Piensa en HTML como el esqueleto o la estructura de una casa. Define dónde están las paredes, las habitaciones, las puertas y las ventanas, pero no se encarga de los colores de las paredes, los muebles o la decoración (eso es trabajo de CSS), ni de la funcionalidad de la casa (eso es JavaScript).

Funciones principales de HTML:

  • Estructura el contenido: Organiza el texto en párrafos, encabezados, listas, tablas, etc.
  • Define el significado (semántica): Le da significado a las diferentes partes del contenido, lo cual es vital para los motores de búsqueda y las tecnologías de asistencia (lectores de pantalla).
  • Inserta contenido multimedia: Permite incrustar imágenes, videos, audio y otros elementos.
  • Crea hipervínculos: La característica más importante de la web; permite enlazar a otras páginas web o recursos.

Sin HTML, no habría estructura en la web. Es el lenguaje que todos los navegadores web entienden para mostrar información.

1.3. Cómo Funciona un Navegador Web.

Un navegador web (como Chrome, Firefox, Edge o Safari) es la aplicación que utilizas para acceder a sitios web en Internet. Pero, ¿cómo funciona realmente?

Cuando introduces una URL (dirección web) en tu navegador y presionas Enter, ocurren varias cosas:

  1. Solicitud al servidor: El navegador envía una solicitud a un servidor web a través de Internet, pidiéndole el archivo HTML de la página que quieres visitar.
  2. Respuesta del servidor: Si el servidor encuentra la página, la envía de vuelta al navegador. Esta respuesta incluye el archivo HTML, junto con otros archivos asociados (CSS, JavaScript, imágenes, etc.).
  3. Procesamiento del HTML (Renderizado):
    • El navegador recibe el archivo HTML y comienza a "leerlo" de arriba a abajo.
    • Construye un modelo de objetos del documento (conocido como DOM - Document Object Model) basado en la estructura HTML.
    • Al mismo tiempo, descarga y procesa los archivos CSS para aplicar los estilos visuales a los elementos HTML.
    • También descarga y ejecuta cualquier código JavaScript para añadir interactividad.
  4. Visualización de la página: Finalmente, el navegador "renderiza" o dibuja la página en tu pantalla, mostrando el contenido estructurado, estilizado e interactivo.

En esencia, el navegador es un intérprete de HTML, CSS y JavaScript, que toma esos lenguajes y los transforma en la representación visual que ves en tu pantalla.

1.4. Herramientas Básicas: Editor de Código (VS Code) y Navegador.

Para empezar a crear tus propias páginas web, solo necesitas dos herramientas fundamentales:

1.4.1. Un Editor de Código

Aunque podrías escribir HTML en un simple Bloc de Notas, un editor de código te facilitará enormemente la vida. Los editores de código están diseñados específicamente para escribir código y ofrecen características como:

  • Resaltado de sintaxis: Colorea diferentes partes del código para que sea más fácil de leer y entender.
  • Autocompletado: Sugiere y completa etiquetas y atributos a medida que escribes.
  • Detección de errores: Te ayuda a identificar problemas en tu código.
  • Extensiones y complementos: Permiten añadir funcionalidades adicionales.

Recomendación: Visual Studio Code (VS Code)

VS Code es un editor de código gratuito, de código abierto y extremadamente popular desarrollado por Microsoft. Es ligero, potente y tiene una vasta comunidad de usuarios y una gran cantidad de extensiones que lo hacen ideal para el desarrollo web. Lo utilizaremos a lo largo de este curso.

Si aún no lo tienes, te recomiendo encarecidamente que lo descargues e instales desde el sitio web oficial: code.visualstudio.com

1.4.2. Un Navegador Web

Necesitarás un navegador web para ver tus páginas HTML. Ya sea Chrome, Firefox, Edge o Safari, cualquiera funcionará. Es buena idea tener al menos dos navegadores instalados, ya que a veces una página puede verse ligeramente diferente en uno que en otro, y esto te ayudará a probar la compatibilidad.

A lo largo de este curso, crearemos archivos HTML en tu editor de código y luego los abriremos directamente en tu navegador para ver los resultados.
