Clase 3: Elementos de Texto I: Encabezados y Párrafos.

Los elementos de texto en HTML son fundamentales para estructurar y presentar el contenido de una página web de manera semántica y legible. En esta sección, repasaremos los encabezados, párrafos, saltos de línea y líneas horizontales.

3.1. Encabezados <h1> a <h6> y su Importancia Semántica.

Los encabezados <h1>, <h2>, <h3>, <h4>, <h5>, <h6> son cruciales para organizar el contenido y tienen una gran importancia semántica. No solo afectan la apariencia visual del texto, sino que también definen la estructura jerárquica del contenido.

3.1.1. Jerarquía y Estructura

El encabezado <h1> debe usarse para el título principal de la página o la sección más importante. Los encabezados subsiguientes (<h2> a <h6>) se utilizan para subtítulos y sub-subtítulos, estableciendo una jerarquía clara en el documento.

3.1.2. Importancia para SEO y Accesibilidad

Los motores de búsqueda utilizan los encabezados para entender la estructura y el tema de tu página, lo que puede mejorar tu posicionamiento. Para los usuarios de lectores de pantalla, los encabezados actúan como puntos de navegación, permitiéndoles comprender rápidamente la estructura del documento.

Ejemplo de Encabezados: 
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Página</title>
</head>
<body>
    <h1>Título Principal de la Página</h1>
    <h2>Sección Importante</h2>
    <h3>Subsección del Tema</h3>
    <h4>Detalle Específico</h4>
</body>
</html>

Recuerda: Evita usar los encabezados solo para dar estilo visual. Su propósito principal es definir la estructura semántica de tu contenido. Tampoco saltes niveles de encabezado (por ejemplo, de <h1> a <h3> directamente) para mantener una jerarquía lógica.

3.2. Párrafos <p>

El elemento <p> se utiliza para agrupar texto en párrafos. Es el elemento de bloque más común para el contenido textual principal y representa un bloque de texto coherente, una unidad de pensamiento.

Ejemplo de Párrafos: 
<p>Este es el primer párrafo de mi documento HTML. Aquí puedo escribir sobre cualquier tema que me interese y necesite.</p>
<p>Este es el segundo párrafo. Los párrafos nos ayudan a organizar el texto en bloques legibles, mejorando la experiencia del usuario.</p>

3.3. Saltos de Línea <br>

El elemento <br> (break) se utiliza para insertar un salto de línea dentro de un bloque de texto. Es un elemento de línea vacío, lo que significa que no tiene contenido ni etiqueta de cierre.

Importante: El <br> debe usarse solo cuando un salto de línea es parte intrínseca del contenido, como en una dirección postal o un poema. No lo uses para crear espacio entre párrafos; para eso, se debe utilizar CSS.

Ejemplo de Uso Correcto de <br>: 
<p>
    Calle Mayor, 123<br>
    Piso 4, Puerta B<br>
    28001 Madrid<br>
    España
</p>

3.4. Líneas Horizontales <hr>.

El elemento <hr> (horizontal rule) representa un cambio temático en el contenido a nivel de párrafo. A menudo, se muestra como una línea horizontal.

Semántica: El <hr> indica una separación temática significativa, como el final de una sección importante o un cambio de escena en una narrativa. No debe usarse solo para dibujar una línea decorativa; para eso, es preferible usar CSS.

Ejemplo de <hr>: 
<p>Aquí termina la introducción a los elementos básicos de HTML.</p>
<hr>
