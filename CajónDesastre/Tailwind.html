Resumen de Tailwind CSS

¿Qué es Tailwind CSS?

Tailwind CSS es un framework de CSS basado en clases utilitarias que permite crear diseños sin escribir CSS personalizado.

Clases esenciales:

  • text-blue-500: Establece el color del texto en azul.
  • bg-gray-200: Aplica un fondo gris claro.
  • p-4: Añade un padding de 1 rem.

Diseño responsive:

  • sm:px-6: Padding horizontal en pantallas pequeñas.
  • lg:text-xl: Tamaño de texto más grande en pantallas grandes.

Flexbox y Grid:

  • flex: Activa el modelo de caja flexible.
  • grid grid-cols-3: Crea una cuadrícula con tres columnas.
