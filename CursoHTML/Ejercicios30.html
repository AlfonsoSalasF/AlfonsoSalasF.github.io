Ejercicio 1: Optimización de Meta Etiquetas y Título

Crea la sección <head> para una página web que trata sobre "Recetas de Cocina Mexicana". Incluye un <title>, una <meta name="description"> relevante y una <meta name="keywords"> (aunque menos importante hoy en día).

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Recetas de Cocina Mexicana Auténtica y Fácil - ¡Prepara tus Favoritos!</title>
    <meta name="description" content="Descubre un sinfín de recetas de cocina mexicana, desde tacos y enchiladas hasta platillos tradicionales. Recetas fáciles y auténticas para todos los niveles.">
    <meta name="keywords" content="recetas mexicanas, cocina mexicana, tacos, enchiladas, comida tradicional, recetas fáciles">
</head>

El título es descriptivo y contiene palabras clave. La descripción es atractiva y resume el contenido. Las palabras clave se incluyen por si acaso, aunque su impacto SEO es mínimo hoy.

Ejercicio 2: Estructura de Encabezados (Jerarquía)

Organiza el siguiente texto utilizando la jerarquía de encabezados (<h1> a <h3>) de forma semántica y correcta.

Consejos para un Jardín Sostenible

Elige Plantas Nativas

Beneficios de las Plantas Nativas

Dónde Comprar Plantas Nativas

Riego Eficiente

Técnicas de Riego por Goteo

Recolección de Agua de Lluvia

<h1>Consejos para un Jardín Sostenible</h1>

<h2>Elige Plantas Nativas</h2>
    <h3>Beneficios de las Plantas Nativas</h3>
    <h3>Dónde Comprar Plantas Nativas</h3>

<h2>Riego Eficiente</h2>
    <h3>Técnicas de Riego por Goteo</h3>
    <h3>Recolección de Agua de Lluvia</h3>

Se usa un solo <h1> para el tema principal. Los <h2> dividen las secciones principales, y los <h3> las subsecciones, manteniendo la jerarquía correcta.

Ejercicio 3: Uso Correcto del Atributo alt y Enlaces

Dado el siguiente HTML, añade el atributo alt a la imagen con una descripción adecuada para SEO y accesibilidad. Además, modifica el enlace para que el texto ancla sea más descriptivo.

<img src="perro-jugando.jpg">
<p>Para saber más, <a href="blog-mascotas.html">haz clic aquí</a>.</p>

<img src="perro-jugando.jpg" alt="Perro golden retriever feliz jugando con una pelota en un parque soleado">
<p>Para saber más, <a href="blog-mascotas.html">visita nuestro blog de consejos para mascotas</a>.</p>

El alt describe la imagen con detalle y potencial palabras clave. El texto ancla del enlace es ahora descriptivo y relevante para el contenido de la página de destino.
