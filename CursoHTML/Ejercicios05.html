Clase 5: Ejercicios de Listas (Ordenadas, Desordenadas y de Definición)

Ejercicios Prácticos de la Clase 5

Para consolidar tus conocimientos sobre listas en HTML, te propongo los siguientes ejercicios. Haz clic en "Resolver" para ver la solución de cada uno.

Ejercicio 1: Crear una Lista Desordenada de Ingredientes

Crea una lista desordenada que represente una lista de ingredientes para una receta simple. No importa el orden.

  • Harina
  • Huevos
  • Azúcar
  • Leche

Solución Ejercicio 1:

<ul>
    <li>Harina</li>
    <li>Huevos</li>
    <li>Azúcar</li>
    <li>Leche</li>
</ul>

Ejercicio 2: Crear una Lista Ordenada de Pasos

Crea una lista ordenada que muestre los pasos para encender un ordenador. El orden es importante.

  • Pulsar el botón de encendido del ordenador.
  • Esperar a que cargue el sistema operativo.
  • Introducir la contraseña (si aplica).
  • Abrir tu navegador web favorito.

Solución Ejercicio 2:

<ol>
    <li>Pulsar el botón de encendido del ordenador.</li>
    <li>Esperar a que cargue el sistema operativo.</li>
    <li>Introducir la contraseña (si aplica).</li>
    <li>Abrir tu navegador web favorito.</li>
</ol>

Ejercicio 3: Lista Ordenada Personalizada

Crea una lista ordenada que comience desde el número 5 y utilice letras mayúsculas como marcador.

  1. Quinto elemento.
  2. Sexto elemento.
  3. Séptimo elemento.

Solución Ejercicio 3:

<ol type="A" start="5">
    <li>Quinto elemento.</li>
    <li>Sexto elemento.</li>
    <li>Séptimo elemento.</li>
</ol>

Ejercicio 4: Lista Ordenada Invertida

Crea una lista ordenada que muestre los números en orden descendente, comenzando desde el 3.

  1. Tercer puesto.
  2. Segundo puesto.
  3. Primer puesto.

Solución Ejercicio 4:

<ol reversed>
    <li>Tercer puesto.</li>>
    <li>Segundo puesto.</li>
    <li>Primer puesto.</li>
</ol>

Ejercicio 5: Crear una Lista de Definición

Crea una lista de definición para los siguientes términos y sus descripciones:

  • CPU: Unidad Central de Procesamiento, el "cerebro" del ordenador.
  • RAM: Memoria de Acceso Aleatorio, memoria volátil para datos temporales.
  • SSD: Unidad de Estado Sólido, dispositivo de almacenamiento más rápido que los HDD.

Solución Ejercicio 5:

<dl>
    <dt>CPU</dt>
    <dd>Unidad Central de Procesamiento, el "cerebro" del ordenador.</dd>
    <dt>RAM</dt>
    <dd>Memoria de Acceso Aleatorio, memoria volátil para datos temporales.</dd>
    <dt>SSD</dt>
    <dd>Unidad de Estado Sólido, dispositivo de almacenamiento más rápido que los HDD.</dd>
</dl>
