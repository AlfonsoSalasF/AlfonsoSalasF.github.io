Clase 31: Ejercicios de Atributos Personalizados de Datos (data-*)

Ejercicios Prácticos de la Clase 31

Estos ejercicios te ayudarán a practicar cómo almacenar y acceder a datos personalizados usando los atributos data-* y JavaScript.

Ejercicio 1: Creación y Acceso Básico a data-*

Crea un botón con el texto "Ver Detalles del Producto". Añade los siguientes atributos data-*:

  • data-product-id con el valor "PROD-001"
  • data-category con el valor "Electrónica"
  • data-price con el valor "499.99"

Luego, escribe un script JavaScript que, al hacer clic en el botón, muestre una alerta con la siguiente información del producto:

"Producto ID: [ID], Categoría: [Categoría], Precio: [Precio]"

HTML y JavaScript (dentro de un <body> o un <script> al final del <body>):

<button id="productDetailsBtn"
        data-product-id="PROD-001"
        data-category="Electrónica"
        data-price="499.99">
    Ver Detalles del Producto
</button>

<script>
    document.addEventListener('DOMContentLoaded', () => {
        const button = document.getElementById('productDetailsBtn');

        button.addEventListener('click', () => {
            const productId = button.dataset.productId;
            const category = button.dataset.category;
            const price = parseFloat(button.dataset.price); // Convertir a número para operar si fuera necesario

            alert(`Producto ID: ${productId}, Categoría: ${category}, Precio: $${price.toFixed(2)}`);
        });
    });
</script>

Este ejercicio demuestra cómo definir los atributos data-* en HTML y cómo acceder a ellos a través de la propiedad .dataset en JavaScript, utilizando la convención camelCase para los nombres de los atributos.

Ejercicio 2: Cambio de Estado de un Elemento

Crea un <div> con el texto "Contenido Oculto". Inicialmente, este <div> tendrá el atributo data-visible="false" y estará oculto con CSS (puedes usar display: none; en el CSS). Añade un botón "Mostrar/Ocultar Contenido".

Al hacer clic en el botón:

  1. Cambia el valor del atributo data-visible entre "true" y "false".
  2. Basado en el nuevo valor de data-visible, alterna la visibilidad del <div> (si está oculto, que se muestre; si está visible, que se oculte).
Contenido Oculto

HTML con CSS y JavaScript:

<style>
    /* CSS para controlar la visibilidad basado en el atributo data-visible */
    #toggleDiv[data-visible="false"] {
        display: none;
    }
    #toggleDiv[data-visible="true"] {
        display: block;
        border: 1px solid #007bff;
        padding: 10px;
        margin-top: 10px;
        background-color: #eaf6ff;
    }
</style>

<div id="toggleDiv" data-visible="false">Contenido Oculto</div>
<button id="toggleBtn">Mostrar/Ocultar Contenido</button>

<script>
    document.addEventListener('DOMContentLoaded', () => {
        const toggleDiv = document.getElementById('toggleDiv');
        const toggleBtn = document.getElementById('toggleBtn');

        toggleBtn.addEventListener('click', () => {
            // Verificar el estado actual
            const isVisible = toggleDiv.dataset.visible === 'true';

            // Alternar el estado
            toggleDiv.dataset.visible = isVisible ? 'false' : 'true';

            // La visibilidad se controla automáticamente con el CSS que reacciona a data-visible
            // Si no usaras CSS, podrías hacer:
            // toggleDiv.style.display = isVisible ? 'none' : 'block';
        });
    });
</script>

Este ejercicio muestra cómo los atributos data-* pueden usarse para gestionar el estado de los elementos de la interfaz de usuario, permitiendo que CSS reaccione a estos estados para cambiar la apariencia, lo cual es una práctica muy común.

Ejercicio 3: Iteración y Extracción de Datos

Crea una lista desordenada (<ul>) de tres ítems. Cada ítem (<li>) debe representar un producto y tener los siguientes atributos data-*:

  • data-product-name (ej. "Laptop", "Smartphone", "Monitor")
  • data-price (ej. "1200", "750", "300")

Escribe un script JavaScript que recorra todos los ítems de la lista y, por cada uno, imprima en la consola (o en un elemento HTML específico) el nombre del producto y su precio.

  • Laptop de alto rendimiento
  • Smartphone de última generación
  • Monitor Ultra HD

HTML y JavaScript:

<ul id="productList">
    <li data-product-name="Laptop" data-price="1200">Laptop de alto rendimiento</li>
    <li data-product-name="Smartphone" data-price="750">Smartphone de última generación</li>
    <li data-product-name="Monitor" data-price="300">Monitor Ultra HD</li>
</ul>
<button id="displayProductsBtn">Mostrar Productos en Consola</button>
<p id="outputProducts"></p>

<script>
    document.addEventListener('DOMContentLoaded', () => {
        const displayProductsBtn = document.getElementById('displayProductsBtn');
        const productList = document.getElementById('productList');
        const outputProducts = document.getElementById('outputProducts');

        displayProductsBtn.addEventListener('click', () => {
            const items = productList.querySelectorAll('li'); // Seleccionar todos los 
  •  dentro de #productList
            outputProducts.innerHTML = 'Productos:<br>'; // Limpiar y añadir título

            items.forEach(item => {
                const name = item.dataset.productName;
                const price = parseFloat(item.dataset.price); // Convertir a número

                // Imprimir en la consola (para desarrolladores)
                console.log(`Producto: ${name}, Precio: $${price.toFixed(2)}`);

                // Mostrar en la página (para el usuario)
                outputProducts.innerHTML += `- ${name}: $${price.toFixed(2)}<br>`;
            });
        });
    });
</script>

    • Este ejemplo demuestra cómo los atributos data-* pueden ser usados para almacenar datos relacionados con cada ítem en una lista, lo que es útil para el filtrado, la ordenación o la visualización dinámica de información sin la necesidad de hacer llamadas adicionales al servidor.
