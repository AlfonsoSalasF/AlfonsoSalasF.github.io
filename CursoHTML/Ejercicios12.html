Clase 12: Ejercicios de Elementos de Entrada de Texto

Ejercicios Prácticos de la Clase 12

Para consolidar tus conocimientos sobre los elementos de entrada de texto en HTML, te propongo los siguientes ejercicios. Intenta resolverlos por ti mismo antes de ver la solución.

Ejercicio 1: Formulario Básico de Perfil de Usuario

Crea un formulario para un perfil de usuario con los siguientes campos de entrada de texto:

  • Un campo para el "Nombre completo" de tipo text.
  • Un campo para el "Correo Electrónico" de tipo email.
  • Un campo para la "Contraseña" de tipo password.
  • Asegúrate de que cada input tenga su atributo name y una label asociada.
<form action="#" method="POST">
    <label for="nombre_completo">Nombre completo:</label>
    <input type="text" id="nombre_completo" name="nombre_completo">

    <label for="email_perfil">Correo Electrónico:</label>
    <input type="email" id="email_perfil" name="email_perfil">

    <label for="password_perfil">Contraseña:</label>
    <input type="password" id="password_perfil" name="password_perfil">

    <button type="submit">Guardar Perfil</button>
</form>

Ejercicio 2: Entrada de Edad con Restricciones

Crea un campo de entrada para la "Edad" utilizando el tipo number. Configura los siguientes atributos:

  • El valor mínimo permitido debe ser 0.
  • El valor máximo permitido debe ser 120.
  • El incremento/decremento debe ser de 1.
  • Añade una label apropiada.
<form action="#" method="POST">
    <label for="edad_ejercicio">Edad:</label>
    <input type="number" id="edad_ejercicio" name="edad_ejercicio" min="0" max="120" step="1">
    <button type="submit">Enviar</button>
</form>

Ejercicio 3: Información de Contacto Adicional

Añade dos campos a un formulario:

  • Uno para el "Número de Teléfono" usando el tipo tel.
  • Otro para el "Sitio Web Personal" usando el tipo url.
  • Incluye sus respectivas labels y atributos name.
<form action="#" method="POST">
    <label for="telefono_ejercicio">Número de Teléfono:</label>
    <input type="tel" id="telefono_ejercicio" name="telefono_ejercicio">

    <label for="website_ejercicio">Sitio Web Personal:</label>
    <input type="url" id="website_ejercicio" name="website_ejercicio">

    <button type="submit">Enviar Contacto</button>
</form>

Ejercicio 4: Barra de Búsqueda con Pista

Crea una barra de búsqueda para un sitio web. Debe cumplir lo siguiente:

  • Utiliza el tipo search.
  • Añade un placeholder que diga "Buscar productos..." para guiar al usuario.
  • Incluye una label apropiada y un atributo name.
<form action="/buscar" method="GET">
    <label for="buscar_productos">Búsqueda de Productos:</label>
    <input type="search" id="buscar_productos" name="q" placeholder="Buscar productos...">
    <button type="submit">Buscar</button>
</form>

Ejercicio 5: Campos de Solo Lectura y Deshabilitados

Crea dos campos de texto:

  • Un campo para "ID de Transacción" con un valor predefinido de "TXN12345" que no pueda ser modificado por el usuario, pero cuyo valor sí se envíe con el formulario.
  • Un campo para "Estado de la Cuenta" con un valor predefinido de "Activa" que no pueda ser modificado por el usuario y cuyo valor no se envíe con el formulario.
  • Usa los atributos readonly y disabled según corresponda.
<form action="#" method="POST">
    <label for="id_transaccion">ID de Transacción:</label>
    <input type="text" id="id_transaccion" name="id_transaccion" value="TXN12345" readonly>

    <label for="estado_cuenta">Estado de la Cuenta:</label>
    <input type="text" id="estado_cuenta" name="estado_cuenta" value="Activa" disabled>

    <button type="submit">Enviar Formulario</button>
</form>

Ejercicio 6: Campo Obligatorio

Crea un formulario con un campo de texto para el "Nombre de Usuario". Este campo debe ser obligatorio para poder enviar el formulario.

  • Asegúrate de que el usuario no pueda enviar el formulario si este campo está vacío.
  • Añade un botón de envío.
<form action="#" method="POST">
    <label for="nombre_usuario_req">Nombre de Usuario (obligatorio):</label>
    <input type="text" id="nombre_usuario_req" name="nombre_usuario_req" required>
    <button type="submit">Registrar</button>
</form>
