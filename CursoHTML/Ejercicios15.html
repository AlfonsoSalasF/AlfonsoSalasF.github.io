Clase 15: Ejercicios de Botones y Campos de Texto Multilínea

Ejercicios Prácticos de la Clase 15

Para consolidar tus conocimientos sobre el uso de `<textarea>` y los diferentes tipos de botones en HTML, te propongo los siguientes ejercicios. Intenta resolverlos por ti mismo antes de ver la solución.

Ejercicio 1: Cuadro de Comentarios Redimensionable

Crea un `<textarea>` que permita al usuario escribir comentarios. Debe tener un tamaño inicial de 4 filas y 50 columnas, y ser redimensionable por el usuario.

<form action="#" method="POST">
    <label for="comentarios">Tus Comentarios:</label>
    <textarea id="comentarios" name="comentarios" rows="4" cols="50"></textarea>
    <button type="submit">Enviar Comentarios</button>
</form>

Ejercicio 2: Campo de Descripción No Redimensionable

Implementa un `<textarea>` para una descripción de producto. Debe tener un tamaño inicial de 6 filas y 40 columnas, y NO debe ser redimensionable por el usuario (ni horizontal ni verticalmente).

  • Necesitarás CSS para controlar el redimensionamiento.
<!-- En el HTML -->
<form action="#" method="POST">
    <label for="descripcion_producto">Descripción del Producto:</label>
    <textarea id="descripcion_producto" name="descripcion_producto" rows="6" cols="40" class="no-resize-textarea"></textarea>
    <button type="submit">Guardar Descripción</button>
</form>

<!-- En tu archivo CSS (ej. formularios.css o un nuevo styles.css) -->
<style>
.no-resize-textarea {
    resize: none;
}
</style>

Ejercicio 3: Botón de Envío Personalizado

Crea un formulario simple con un campo de texto para el nombre y un botón de envío. El botón de envío debe ser un `<button>` en lugar de un `<input type="submit">`, y su texto debe decir "Registrarse Ahora".

<form action="/registro" method="POST">
    <label for="nombre_registro">Nombre:</label>
    <input type="text" id="nombre_registro" name="nombre_registro" required>
    <button type="submit">Registrarse Ahora</button>
</form>

Ejercicio 4: Botón de Restablecimiento de Formulario

Crea un formulario con un campo de email prellenado y un botón para restablecer el formulario a su estado inicial. Utiliza un `<input type="reset">` y un `<button type="reset">` para mostrar ambos enfoques.

<form action="#" method="POST">
    <label for="email_contacto">Email:</label>
    <input type="email" id="email_contacto" name="email_contacto" value="ejemplo@dominio.com">
    <input type="reset" value="Limpiar (Input Reset)">
    <button type="reset">Limpiar (Button Reset)</button>
</form>

Ejercicio 5: Botón Genérico para Script

Crea un botón que no envíe un formulario ni lo restablezca, sino que esté diseñado para activar una función JavaScript. Su texto debe ser "Generar Reporte" y su `type` debe ser el apropiado.

<!DOCTYPE html>
<html>
<head>
    <title>Ejemplo Botón Genérico</title>
</head>
<body>
    <button type="button" onclick="generarReporte()">Generar Reporte</button>

    <script>
        function generarReporte() {
            alert('¡Reporte generado! (Esta es una simulación JavaScript)');
            console.log('Se ha activado la función de generación de reporte.');
        }
    </script>
</body>
</html>
