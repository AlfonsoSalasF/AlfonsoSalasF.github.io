Clase 20: Tablas HTML: Organización de Datos Tabulares.

Las tablas HTML son elementos fundamentales para organizar y mostrar datos en formato tabular, es decir, en filas y columnas. Aunque no deben usarse para el diseño de la página (para eso usamos CSS y Flexbox/Grid), son indispensables para presentar información estructurada como estadísticas, listados de productos, horarios, etc.

20.1. La etiqueta <table>.

El elemento <table> es el contenedor principal de toda la tabla. Dentro de él, se definen las filas, encabezados y celdas de datos.

Ejemplo básico:

<table>
    <caption>Productos Destacados</caption>
    <tr>
        <th>Producto</th>
        <th>Precio</th>
    </tr>
    <tr>
        <td>Laptop</td>
        <td>1200€</td>
    </tr>
    <tr>
        <td>Teléfono</td>
        <td>800€</td>
    </tr>
</table>
Productos Destacados
Producto Precio
Laptop 1200€
Teléfono 800€

<caption>: Aunque no se menciona en la sección, la etiqueta <caption> proporciona un título o una breve descripción para toda la tabla, lo cual es muy importante para la accesibilidad.

20.2. Filas y columnas: <tr>, <th>, <td>.

Estos son los elementos fundamentales para construir la estructura de la tabla:

  • <tr> (Table Row): Define una fila de celdas en una tabla. Cada <tr> representa una nueva fila.
  • <th> (Table Header): Define una celda de encabezado en una tabla. El texto dentro de <th> suele aparecer en negrita y centrado por defecto, y semánticamente indica que es el encabezado de una columna o fila.
  • <td> (Table Data): Define una celda de datos estándar en una tabla. Contiene el contenido real de la tabla.

Ejemplo: Horario de Clases.

<table>
    <caption>Horario Semanal</caption>
    <tr>
        <th>Día</th>
        <th>Mañana</th>
        <th>Tarde</th>
    </tr>
    <tr>
        <td>Lunes</td>
        <td>Matemáticas</td>
        <td>Historia</td>
    </tr>
    <tr>
        <td>Martes</td>
        <td>Ciencias</td>
        <td>Lengua</td>
    </tr>
</table>
Horario Semanal
Día Mañana Tarde
Lunes Matemáticas Historia
Martes Ciencias Lengua

20.3. Encabezado, cuerpo y pie de tabla: <thead>, <tbody>, <tfoot>.

Para mejorar la semántica y la organización de tablas complejas, HTML proporciona elementos para dividir la tabla en secciones lógicas. Esto es muy útil para la accesibilidad y para aplicar estilos específicos a cada parte de la tabla.

  • <thead> (Table Head): Contiene el contenido de encabezado de la tabla (normalmente las etiquetas <th>).
  • <tbody> (Table Body): Contiene el cuerpo principal de los datos de la tabla. Puede haber múltiples <tbody> en una tabla.
  • <tfoot> (Table Foot): Contiene el contenido del pie de la tabla (ej., totales, notas, resúmenes). Aunque semánticamente es el pie, visualmente puede aparecer antes del <tbody> en el HTML, pero se renderizará al final de la tabla.

Ejemplo: Informe de Ventas Mensuales.

<table>
    <caption>Informe de Ventas por Producto (Junio)</caption>
    <thead>
        <tr>
            <th>Producto</th>
            <th>Cantidad Vendida</th>
            <th>Ingresos (€)</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Camiseta</td>
            <td>150</td>
            <td>3000</td>
        </tr>
        <tr>
            <td>Pantalón</td>
            <td>80</td>
            <td>4000</td>
        </tr>
        <tr>
            <td>Zapatillas</td>
            <td>50</td>
            <td>5000</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td>Total</td>
            <td>280</td>
            <td>12000</td>
        </tr>
    </tfoot>
</table>
Informe de Ventas por Producto (Junio)
Producto Cantidad Vendida Ingresos (€)
Camiseta 150 3000
Pantalón 80 4000

Total 280 12000

20.4. Combinar celdas: rowspan y colspan.

Los atributos rowspan y colspan se utilizan para hacer que una celda ocupe el espacio de múltiples filas o columnas, respectivamente. Son esenciales para crear tablas con diseños más complejos.

  • colspan: Especifica el número de columnas que una celda debe abarcar.
  • rowspan: Especifica el número de filas que una celda debe abarcar.

Ejemplo: Horario con celdas combinadas.

<table>
    <caption>Horario Detallado</caption>
    <thead>
        <tr>
            <th>Día</th>
            <th colspan="2">Mañana</th>
            <th>Tarde</th>
        </tr>
        <tr>
            <th></th> <!-- Celda vacía para alinear -->
            <th>9:00 - 11:00</th>
            <th>11:00 - 13:00</th>
            <th>15:00 - 17:00</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Lunes</td>
            <td>Matemáticas</td>
            <td>Física</td>
            <td rowspan="2">Deportes</td> <!-- Ocupa 2 filas -->
        </tr>
        <tr>
            <td>Martes</td>
            <td colspan="2">Programación</td> <!-- Ocupa 2 columnas -->
        </tr>
    </tbody>
</table>
Horario Detallado
Día Mañana Tarde
9:00 - 11:00 11:00 - 13:00 15:00 - 17:00
Lunes Matemáticas Física Deportes
Martes Programación

Cuidado con el abuso: Combinar celdas puede hacer que la estructura de la tabla sea más compleja y, en algunos casos, menos accesible. Úsalo solo cuando sea estrictamente necesario para la presentación lógica de los datos.
