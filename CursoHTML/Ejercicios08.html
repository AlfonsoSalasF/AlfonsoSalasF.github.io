Ejercicio 1: Estructura Básica con Semánticos

Crea la estructura básica de una página web utilizando las etiquetas semánticas <header>, <nav>, <main> y <footer>. Dentro de <header>, incluye un <h1> con el título "Mi Sitio Web Semántico". Dentro de <nav>, crea una lista desordenada con dos enlaces: "Inicio" y "Acerca de". Dentro de <main>, un <p> con "Contenido principal de la página". Y en <footer>, un párrafo con "Copyright © 2025".

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Sitio Web Semántico</title>
</head>
<body>
    <header>
        <h1>Mi Sitio Web Semántico</h1>
        <nav>
            <ul>
                <li><a href="#">Inicio</a></li>
                <li><a href="#">Acerca de</a></li>
            </ul>
        </nav>
    </header>
    <main>
        <p>Contenido principal de la página.</p>
    </main>
    <footer>
        <p>Copyright &copy; 2025</p>
    </footer>
</body>
</html>

Ejercicio 2: Uso de <article>, <section> y <aside>

Dentro de la etiqueta <main> (del ejercicio anterior o de una estructura básica), crea un <article> para un post de blog con un título (<h2>) "Mi Primer Post Semántico" y un párrafo de contenido. Dentro de este <article>, añade una <section> para "Comentarios" con un <h3>. Fuera del <article> pero aún dentro de <main>, añade una <aside> con un <h3> "Artículos Relacionados" y una lista desordenada de enlaces a otros posts (inventados).

<main>
    <article>
        <h2>Mi Primer Post Semántico</h2>
        <p>Este es el contenido de mi post de blog, que es una pieza de contenido independiente.</p>
        <section>
            <h3>Comentarios</h3>
            <p>Sé el primero en comentar este artículo.</p>
        </section>
    </article>
    <aside>
        <h3>Artículos Relacionados</h3>
        <ul>
            <li><a href="#">HTML5 para Principiantes</a></li>
            <li><a href="#">CSS Básico</a></li>
        </ul>
    </aside>
</main>
